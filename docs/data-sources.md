## Enrichment

- Public Dianping search currently returns a login / risk-control page without a durable authenticated browser session, so it is not used as a deterministic source.
- AMap Place Text (`/v3/place/text`) enriches the MICHELIN list with local POI matches: per-person cost when `biz_ext.cost` exists and passes the local data-quality range, top dish/tag labels from `tag`/`atag`, local cover photos from `photos`, and stored marker coordinates.
- Restaurant cover images prefer the MICHELIN list card when exposed by the listing page and fall back to AMap POI photos.

## Notes

- The official Michelin Guide site exposes public listing pages, but no stable public restaurant-list API was confirmed during this build.
- The checked-in data was generated by `scripts/fetch-michelin-data.mjs` on 2026-04-29 from live MICHELIN listing pages plus AMap enrichment.
- The four-city data currently contains 490 restaurants: Xiamen 42, Shanghai 154, Chengdu 76, Hong Kong 218.
- The live Shanghai listing page reports 155 restaurants, but one listed card is a Hong Kong branch mixed into "surroundings"; it is excluded by city field. The live Hong Kong listing page reports 218 restaurants, while the corporate 2026 press release says 219 Hong Kong restaurants.
- Per-person price and signature-dish values are enrichment fields. Missing or outlier values stay empty rather than being invented from MICHELIN price bands.
- Gaode / AMap JSAPI 2.0 is the map surface. The app first reads `/amap-config.json`, then falls back to `VITE_AMAP_KEY` and `VITE_AMAP_SECURITY_CODE`, and initializes the map with `mapStyle: "amap://styles/whitesmoke"`.
- Marker coordinates are stored in the local restaurant data after MICHELIN list scraping and AMap REST Place Text resolution, so marker rendering does not depend on runtime `PlaceSearch` permission.
- AMap `PlaceSearch` remains a secondary resolver for future restaurants that do not yet have stored coordinates.
- Without a usable key, local verification still renders a nonblank white-smoke map surface while live Gaode tiles stay unavailable.
