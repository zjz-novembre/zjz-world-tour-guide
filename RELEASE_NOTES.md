# Dine Guide Release Notes

## 1.1.1-202605252212

Refreshes Dine Guide's cross-guide switching identity and custom map styling. The Michelin-to-Black-Pearl switch now uses the provided Black Pearl Restaurant Guide mark on both web and iOS, web maps use the published custom AMap style `amap://styles/653152ed110317541d8e1dabfa110aae`, Black Pearl web pins are no longer washed by the 24% white-smoke overlay, and the iOS bundle carries the updated native AMap style data package.

## 1.1.0-202605252101

Applies a custom minimalist AMap base layer to the native iOS experience, aligning the map surface with Dine Guide's restrained white-smoke visual system while preserving live road context. Michelin map markers now receive a controlled 24% white tonal wash to match the web map presentation without altering Black Pearl marker treatment.
